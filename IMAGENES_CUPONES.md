## 🔄 Cómo Cambiar Imágenes

### Formato de URL
```
https://images.unsplash.com/photo-XXXXXX?w=800&h=400&fit=crop
```

### Parámetros Recomendados
- `w=800` - Ancho óptimo para cupones
- `h=400` - Alto óptimo (ratio 2:1)
- `fit=crop` - Recorta para ajustar exactamente

### Buscar Más Imágenes

1. Visita [Unsplash.com](https://unsplash.com)
2. Busca términos como:
   - "fitness person"
   - "running athlete"
   - "gym workout"
   - "healthy nutrition"
   - "outdoor adventure"
   - "smartwatch fitness"
3. Copia el ID de la foto (ej: `photo-1234567890abc`)
4. Construye la URL completa

## ✨ Tips de Selección

### ✅ Buenas Imágenes
- **Personas reales:** Conecta emocionalmente
- **Acción/movimiento:** Transmite energía
- **Buena iluminación:** Natural, clara
- **Fondo no muy ocupado:** Resalta el mensaje
- **Alta calidad:** Mínimo 1600x800px

### ❌ Evitar
- Imágenes muy oscuras
- Demasiado texto en la imagen
- Fondos muy saturados
- Imágenes genéricas sin emoción
- Baja resolución
